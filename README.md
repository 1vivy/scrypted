 * Google Home
 * Apple HomeKit
 * Amazon Alexa

Supported accessories: 
 * https://github.com/koush/scrypted/tree/main/plugins

## Installation


```sh
git clone https://github.com/koush/scrypted.git
cd scrypted/server
npm install
npm run serve
```
